Application Settings
Important Notice

In this page, you can view and manage the settings of your application. You can enable or disable features such as Automated Data Backups, Automated Software Updates and Bug Fixes, as well as Automated Model Updates and Monitoring. This feature is currently under development, and will be available in the next releases.

Feature Info Enabled
Automated Data Backups Automated Data Backups is a process of copying, archiving, and storing data so that it can be restored in case of data loss. This is to protect the privacy and security of your data by ensuring that it is backed up regularly to prevent data loss due to accidental deletion, corruption, or hardware failure. We highly recommend enabling this feature to ensure the safety and security of your data.
Automated Software Updates & Bug Fixes Automated Software Updates & Bug Fixes is a process of automatically updating the software and fixing bugs to ensure that the application is up-to-date and secure. This is to protect the privacy and security of your data by ensuring that the application is always running the latest version with the latest security patches and bug fixes. We highly recommend enabling this feature to ensure the safety and security of your data.
Automated Model Updates & Monitoring Automated Model Updates & Monitoring is a process of automatically updating the machine learning models and monitoring their performance to ensure that they are accurate and up-to-date. This is to protect the privacy and security of your data by ensuring that the models are always running the latest version with the latest training data and parameters. We highly recommend enabling this feature to ensure the accuracy and reliability of your models.
