Create Assistant Memory
Tip

In this page, you can create a new assistant memory. Assistant memories are an essential step to optimize an assistant, since they can be helpful in avoiding certain common errors and ensuring that the responses follow the requirements you would like to set. Fill in the form with the assistant memory's information and click on the "Create" button to save the assistant memory. A user-specific memory is a memory that is specific to a user, in which the memory will only be accessible for the chats of that user. An assistant-specific memory is a memory that is specific to the assistant, in which the memory will be accessible for all the chats of the assistant, independent of the user.
